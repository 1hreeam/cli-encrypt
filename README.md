# @1hreeam/cli-encrypt

CLI tool to encrypt and decrypt files.

## How to use?

```bash
cli-encrypt -i <input-path> -o <output-path>
```

## Upcoming features

- Logging modes (`--verbose` `--silent`)
- Deafult output file naming (`.enc`, `.dec`)
- Large files support (rn reading the whole file into memory)
- Compression
